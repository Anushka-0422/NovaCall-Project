# 🚀 NovaCall – Video Conferencing & Live Chat Platform

NovaCall is a full-stack real-time communication platform that enables users to connect through video conferencing and live chat. It is designed with a scalable architecture using separate frontend and backend systems.

---

## 🌟 Features

- 🎥 Real-time video conferencing *(in progress)*  
- 💬 Live chat functionality  
- ⚡ Fast frontend using Vite  
- 🔗 REST API-based backend  
- 📡 Real-time communication (Socket.io/WebRTC – planned)  
- 📱 Responsive UI  

---

## 🛠️ Tech Stack

### 💻 Frontend
- HTML, CSS, JavaScript  
- Vite React 

### ⚙️ Backend
- Node.js  
- Express.js  

### 🗄️ Database
- MongoDB / MySQL

## ⚙️ Installation & Setup

1. Clone the repository:
git clone : https://github.com/your-username/novacall.git

2. Setup Backend

npm install
npm start

4. Setup Frontend
cd frontend
npm install
npm run dev
